Raw behaviour / emotion (HTTP → WebSocket)

POSTs to the bridge are broadcast as JSON to every WebSocket client (your ESP32). Use this to exercise VerbSystem / EmotionSystem without agent lifecycle events. Requires the same Authorization: Bearer token as other bridge APIs. The bridge sends permissive CORS headers so you can serve this folder from any local port (e.g. npx http-server) while the API stays on 127.0.0.1:8787.
